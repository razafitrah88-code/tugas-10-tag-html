# tugas-10-tag-html

1. DOCTYPE html
Ini sebenarnya bukan “tag” seperti yang lain, tapi deklarasi di paling atas file. Fungsinya untuk memberi tahu browser bahwa halaman ini menggunakan standar HTML5. Tanpa ini, browser bisa masuk ke mode lama (quirks mode) yang bikin tampilan jadi berantakan atau tidak konsisten antar browser. Jadi ini penting banget walaupun cuma satu baris.


2. html
Tag ini adalah akar dari seluruh halaman web. Semua elemen HTML harus berada di dalamnya. Bisa dibilang ini seperti “wadah besar” yang menampung semua struktur halaman. Biasanya juga digunakan untuk menentukan bahasa halaman (misalnya bahasa Indonesia) agar mesin pencari dan browser lebih paham isi konten.


3. head
Bagian ini berisi informasi tentang halaman yang tidak langsung terlihat oleh pengguna. Misalnya judul halaman, pengaturan karakter huruf, atau link ke file CSS dan JavaScript. Walaupun tidak terlihat, bagian ini sangat penting karena mengatur bagaimana halaman ditampilkan dan dikenali oleh browser maupun mesin pencari.


4. title
Tag ini ada di dalam head dan berfungsi untuk menentukan judul halaman yang muncul di tab browser. Selain itu, title juga penting untuk SEO (Search Engine Optimization) karena digunakan oleh mesin pencari seperti Google untuk menampilkan nama halaman di hasil pencarian.


5. body
Ini adalah bagian utama yang berisi semua konten yang dilihat oleh pengguna. Semua teks, gambar, video, tombol, dan elemen visual lainnya berada di sini. Kalau diibaratkan, head itu “otak pengaturan”, sedangkan body itu “isi yang dilihat langsung”.


6. h1
Tag ini digunakan untuk membuat judul atau heading. h1 adalah tingkat paling tinggi dan paling besar, biasanya dipakai untuk judul utama halaman. Selain untuk tampilan, heading juga membantu struktur konten supaya lebih rapi dan mudah dipahami, baik oleh pengguna maupun mesin pencari.


7. p
Tag ini digunakan untuk membuat paragraf. Dengan adanya p, teks jadi lebih terstruktur dan tidak berantakan. Browser juga otomatis memberikan jarak antar paragraf sehingga tulisan lebih nyaman dibaca.


8. a
Tag ini digunakan untuk membuat hyperlink atau link. Link ini bisa mengarah ke halaman lain, file, atau bahkan bagian tertentu dalam halaman yang sama. Ini adalah salah satu elemen paling penting di web karena memungkinkan pengguna berpindah dari satu halaman ke halaman lain dengan mudah.


9. img
Tag ini digunakan untuk menampilkan gambar di halaman web. Gambar sangat penting untuk membuat tampilan lebih menarik dan informatif. Biasanya juga dilengkapi dengan teks alternatif (alt) yang berguna jika gambar gagal dimuat atau untuk membantu aksesibilitas.


10. ul dan li
ul digunakan untuk membuat daftar yang tidak berurutan (biasanya ditandai dengan bullet/titik), sedangkan li digunakan untuk setiap item dalam daftar tersebut. Elemen ini membantu menyusun informasi agar lebih rapi dan mudah dipahami, terutama jika berisi banyak poin.
